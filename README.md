# VogueVista

VogueVista is a React-based e-commerce application designed for a seamless shopping experience, featuring categories for men's, women's, and kids' fashion. The platform allows users to browse products, manage their shopping cart, and view product details with ease.

## Features

- **Category Navigation**: Main shop page with dedicated sections for Men's, Women's, and Kids' fashion.
- **Shopping Cart Functionality**: Easily add and remove items from the cart.
- **Product Display**: Attractive product listings with clear images and descriptions.
- **Dynamic Cart Summary**: Total cart amount updates automatically as items are added or removed.

## Technologies Used

- **Frontend**: React.js
- **Styling**: CSS/Tailwind CSS
- **State Management**: React Hooks
