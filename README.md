# VogueVista

VogueVista is a fashion website built with React, designed to provide a seamless and engaging shopping experience. The website includes various features to help users browse and purchase fashion items effortlessly.

## Features

1. **Home Page**
   - Showcases popular dresses and new collections to attract users' attention and highlight trending items.

2. **Women, Men, Kids Sections**
   - Dedicated sections for women, men, and kids to make browsing specific categories easy and convenient.

3. **Add to Cart Page**
   - Displays detailed information about dresses, including different sizes, descriptions, and reviews. Users can select their desired size and add items to their cart.

4. **Login Page**
   - Allows users to securely log in to their accounts to manage their shopping experience and access personalized features.

5. **Cart Totals Page**
   - Shows the items added to the cart, with the ability to delete items or adjust their quantities. The total price is automatically updated as items are added or removed.

# Technologies Used
- React: JavaScript library for building user interfaces.
- CSS: Styling the website.
- HTML: Markup language for structuring the web pages.
